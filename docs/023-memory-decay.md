Memory Decay
Purpose

Mnemosyne should not remember everything forever.

Memories that are important should persist.

Memories that are trivial should gradually weaken over time.

Philosophy

Forgetting is compression.

Not destruction.

Importance Levels
Importance 1-3

Temporary memories.

Examples:

Weather
One-time reminders
Random jokes

Decay speed:

Fast

Importance 4-6

Useful memories.

Examples:

Favorite foods
TV shows
Daily routines

Decay speed:

Moderate

Importance 7-8

Long-term memories.

Examples:

Hobbies
Close friendships
Career goals

Decay speed:

Very slow

Importance 9-10

Core memories.

Examples:

Relationship with Mimi
Family members
Birthdays
Life values
Major events

Decay speed:

Never

Reinforcement

Repeated access strengthens memories.

Example:

Access count: 1

Importance: 5

↓

Access count: 10

Importance: 8

Inactivity

Memories that are never accessed slowly lose importance.

Example:

Last accessed
↓

Decay function
↓

Reduced score

States

A memory can exist in one of four states:

Active

Normal memory.

Dormant

Rarely used.

Still recoverable.

Archived

Hidden from normal retrieval.

Can be restored.

Deleted

Permanent removal.

Deletion should be extremely rare.

Pinning

Pinned memories never decay.

Examples:

Family
Partners
Life values
Birthdays
Principle

Humans forget details.

They remember meaning.

Mnemosyne should do the same.
