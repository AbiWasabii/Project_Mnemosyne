# Memory Search

## Purpose

Find relevant memories efficiently.

---

## Philosophy

Retrieval quality matters more than storage quantity.

---

## Search Types

### Exact Search

Search for precise facts.

Example:

"What is Mimi's birthday?"

---

### Semantic Search

Search for meaning.

Example:

"What drinks does Mimi enjoy?"

↓

"Mimi likes matcha."

---

### Temporal Search

Search by time.

Examples:

* Last week
* Last month
* 2026

---

### Person Search

Search by owner.

Examples:

* Alex
* Mimi
* Shared

---

### Tag Search

Search using categories.

Examples:

* travel
* hobbies
* food

---

### Importance Search

Retrieve high-value memories first.

---

## Ranking

Priority:

1. Similarity
2. Importance
3. Confidence
4. Recency
5. Frequency

---

## Limits

Default:

Top 5 memories.

Maximum:

Top 20 memories.

---

## Principle

Find the right memories.

Not all memories.
