# Memory Conflicts

## Purpose

Resolve contradictions without fabricating information.

---

## Example

Memory A:

"Mimi likes green tea."

Importance:

6

Date:

2025

---

Memory B:

"Mimi prefers matcha."

Importance:

8

Date:

2026

---

Resolution:

Prefer Memory B.

Reason:

Higher importance and newer information.

---

## Conflict Priority

### 1. Explicit correction

Highest priority.

Example:

"I used to like coffee, but now I prefer tea."

---

### 2. Importance

More important memories win.

---

### 3. Recency

Newer memories override older memories.

---

### 4. Frequency

Frequently reinforced memories are preferred.

---

### 5. User uncertainty

If confidence is low:

Return multiple possibilities.

Example:

"Past conversations suggest Mimi enjoys green tea, although more recent memories indicate matcha may now be preferred."

---

## Never

Never merge contradictory facts.

Never invent missing details.

Never hide uncertainty.

---

## Principle

Contradictions are signals.

Not errors.
