# Project Mnemosyne

A private, local-first digital brain for Alex and Mimi.

---

## Vision

Mnemosyne is a shared memory and intelligence system designed to preserve knowledge, experiences, and memories across decades.

Built with:

- Reliability
- Privacy
- Modularity
- Long-term maintainability

---

## Principles

- Models are replaceable.
- Memories are sacred.
- Simplicity over complexity.
- Tools are optional.
- Never fabricate memories.
